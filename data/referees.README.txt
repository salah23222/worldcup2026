data/referees.json — قائمة حكام كأس العالم 2026 (تحرير يدوي)
============================================================

لا يوجد API مجاني موثوق يعطي قائمة حكام مونديال 2026 الرسمية أو تقييماتهم.
لذلك هذا الملف يُملأ يدوياً من مالك الموقع عندما يعلن FIFA أسماء الحكام.

الملف الحالي = مصفوفة فارغة [] حتى لا يظهر أي اسم وهمي.
عند إعلان FIFA، استبدل [] بقائمة من الكائنات بهذا الشكل بالضبط:

[
  {
    "name": "Szymon Marciniak",   ← اسم الحكم كما يظهر في $m['referee'] (مطابقة حرفية)
    "country_ar": "بولندا",        ← اسم الدولة بالعربية (للعرض)
    "country_en": "Poland",        ← اسم الدولة بالإنجليزية (للعرض)
    "flag": "pl"                    ← رمز ISO ثنائي صغير (يجلب العلم من flagcdn.com)
  },
  {
    "name": "...",
    "country_ar": "...",
    "country_en": "...",
    "flag": "..."
  }
]

ملاحظات:
- حقل "name" يجب أن يطابق تماماً قيمة $m['referee'] في بيانات المباراة
  حتى يُحسب عدد المباريات التي أدارها الحكم تلقائياً (matchesOfficiated).
- لا يوجد حقل تقييم (rating): لا مصدر مجاني له. صفحة الحكام تعرض "—".
  إن رغب المالك بإضافة تقييم يدوي مستقبلاً، يمكن إضافة حقل "rating".
- بعد التعديل احفظ الملف بترميز UTF-8 وتأكد أنه JSON صالح.
